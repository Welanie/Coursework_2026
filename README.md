# ChemCards

Курсовая работа по веб-программированию.

Тема: **карточки для запоминания химических формул**.

Сайт сделан как небольшой тренажер по химии. Пользователь видит формулу вещества, вводит название и сразу получает результат. После тренировок сохраняется прогресс, статистика и рейтинг.

## Что используется

- PHP;
- MySQL;
- PDO;
- HTML;
- CSS;
- немного JavaScript;
- OpenServer для запуска на компьютере.

## Как запустить

1. Скопировать папку проекта в домен OpenServer, например:

```text
C:\OSPanel\home\chemcards.local\public
```

2. Открыть phpMyAdmin.

3. Создать базу данных:

```text
chem_flashcards
```

4. Импортировать файл:

```text
sql/database.sql
```

5. Проверить настройки подключения в файле:

```text
includes/config.php
```

Обычные настройки для OpenServer:

```text
host: 127.0.0.1
database: chem_flashcards
user: root
password: пустой
```

Если в OpenServer используется MySQL-8.0, в `DB_HOST` можно указать:

```text
MySQL-8.0
```

6. Открыть сайт в браузере:

```text
http://chemcards.local
```

## База данных

В проекте используются таблицы:

- `users` — пользователи сайта;
- `categories` — категории веществ;
- `flashcard_sets` — наборы карточек;
- `flashcards` — карточки с формулами;
- `user_progress` — прогресс пользователя;
- `study_sessions` — результаты тренировок;
- `favorites` — избранные карточки.

SQL-файл находится здесь:

```text
sql/database.sql
```

Он создает таблицы и добавляет тестовые данные.
