# ChemCards

Курсовая работа по веб-программированию, вариант 8: **«Карточки для запоминания химических формул»**.

Приложение помогает учить химические формулы и названия веществ: пользователь видит формулу, вводит название, получает проверку ответа, а система сохраняет прогресс и статистику.

## Технологии

- PHP 7.4+;
- MySQL / MariaDB;
- PDO;
- HTML, CSS, JavaScript;
- сессии PHP;
- OpenServer для локального запуска.

## Установка

1. Скопировать проект в папку домена OpenServer.
2. Создать базу данных и импортировать файл `sql/database.sql` через phpMyAdmin.
3. Проверить настройки подключения в `includes/config.php`.
4. Открыть проект в браузере.

По умолчанию используются настройки:

```text
host: 127.0.0.1
database: chem_flashcards
user: root
password: пустой
```

Если проект открывается не из корня домена, задайте `BASE_URL` в `includes/config.php` или через переменную окружения.

## Реализованные функции

- регистрация, вход, выход;
- роли `guest`, `user`, `admin`;
- публичные и пользовательские наборы карточек;
- каталог карточек с поиском, фильтром и пагинацией;
- детальная страница карточки;
- обычный режим обучения;
- режим повторения проблемных карточек;
- тестовый режим на 10 карточек;
- проверка ответа через нормализацию строки;
- сохранение правильных и неправильных ответов;
- рейтинг карточки и дата следующего повторения;
- график прогресса по учебным сессиям;
- избранные карточки;
- глобальный рейтинг пользователей и лидерборд на главной странице;
- пользовательский CRUD для собственных наборов и карточек;
- админ-панель: CRUD наборов, CRUD карточек, управление пользователями;
- SQL-скрипт создания и наполнения БД.

## Основные таблицы

- `users` — пользователи, роли и рейтинговые очки;
- `categories` — категории веществ;
- `flashcard_sets` — наборы карточек;
- `flashcards` — формулы и названия веществ;
- `user_progress` — прогресс пользователя по каждой карточке;
- `study_sessions` — результаты тренировок и тестов;
- `favorites` — избранные карточки.
