# Чек-лист тестирования

## Авторизация

- Зарегистрировать нового пользователя.
- Проверить, что пользователь появился в таблице `users`.
- Войти с правильным паролем.
- Проверить ошибку при неправильном пароле.
- Выйти из аккаунта.

## Права доступа

- Открыть админку под `admin`.
- Убедиться, что обычный пользователь не открывает `/admin/`.
- Проверить, что гость видит только публичные наборы.

## Карточки и наборы

- Открыть список наборов.
- Найти набор через поиск.
- Открыть карточку.
- Проверить поиск по формуле.
- Проверить фильтр по категории.
- Проверить пагинацию каталога.

## CRUD

- В админке создать набор.
- Изменить название набора.
- Добавить карточку.
- Изменить карточку.
- Удалить карточку.
- Удалить тестовый набор.

## Бизнес-логика

- Запустить обычную тренировку.
- Ввести правильный ответ.
- Ввести неправильный ответ.
- Убедиться, что неправильная карточка появляется повторно.
- Запустить тестовый режим.
- Завершить тест и проверить результат.
- Запустить режим повторения.

## Дополнительные модули

- Добавить карточку в избранное.
- Удалить карточку из избранного.
- Создать пользовательский набор.
- Добавить карточки в пользовательский набор.
- Проверить график прогресса после нескольких тренировок.

## Адаптивность

- Проверить главную страницу на ширине 390 px.
- Проверить таблицы на мобильной ширине.
- Проверить формы входа, регистрации и тренировки.
